Console Project

Visual Studio Code is a freeware source-code editor made by Microsoft for Windows, Linux and macOS. Features include support for debugging, syntax highlighting, intelligent code completion, snippets, code refactoring, and embedded Git. To set it up- go to code.visualstudio.com, click on the the download button at the top right, select your OS and the installer gets downloaded.

Web developers often log messages to the Console to make sure that their JavaScript is working as expected. They log these messages on a screen called the console. to open console- right click on the web page, select the 'inspect' option from the shorctcut menu and select the console tab on the inspect pane. to write something on the console, use the console.log() function.
